Kotlin基础（二）

Kotlin基础–函数

定义

函数使用 fun 关键字来定义, 使用 Unit 来代表空值返回

fun sum(x:Int, y:Int):Int{
    return x+y
}

参数

1、默认参数 ： 默认参数被省略后，会自动使用默认值
fun sum(x:Int = 0 , y:Int):Int{
    return x+y
}
sum(y=2) // 默认x=0 进行运算 , 这种 “参数名=参数值” 的调用使用了命名参数，避免了参数过多时，无法轻易的分辨出每个参数的具体含义；

2、可变参数 vararg , 在kotlin中 ，可变参数实际通过数组来处理， 特别的，在kotlin中操作数组时，需要使用伸展操作符（*）
fun sum(vararg paras:Int):Int{
    var result:Int =0
    for(para in paras) result+=para
    return result
}
val paraArray = intArrayOf(1,2,3,4)
sum(*paraArray)

单表达式函数

即函数返回了单个的表达式，如：
fun sum(x:Int = 0 , y:Int):Int{
    return x+y
}
等价于 fun sum(x:Int = 0 , y:Int):Int = x+y
等价于 fun sum(x:Int = 0 , y:Int) = x+y // x,y类型已经声明，那么返回类型可推导

成员函数

定义在类或对象的内部的函数

局部函数

定义在函数内部的函数，可以访问其外部函数(即闭包)的成员变量，使用时也要由外部函数来调用

// 成员函数
fun outMethod(x: Int, y: Int) {
    val memberVal = 10
    // 局部函数
    fun innerMethod(z: Double): Double {
        return z * memberVal
    }
}

匿名函数

没有函数名
fun(x:Int = 0 , y:Int):Int{
    return x+y
}

Lambda 表达式

本质上是一个匿名函数，而且可作为一个接口被实现
val sum = {x:Int ,y:Int -> x+y }

高阶函数

将函数作为参数或者返回值的函数
fun sum(x:Int = 0 , y:Int, paraFunc:(a:Int,b:Int)->Boolean):Int{
    return if(paraFunc(x,y)) x+y else x-y
}

在Kotlin中，如果函数最后一个参数接收函数，那么这个函数可以放在外面
fun sum(x:Int = 0 , y:Int, paraFunc:(a:Int,b:Int)->Boolean):Int{
    return if(paraFunc(x,y)) x+y else x-y
}
