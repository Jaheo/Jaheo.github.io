Kotlin基础（二）

Kotlin基础–函数

定义

函数使用 fun 关键字来定义, 使用 Unit 来代表空值返回

fun sum(x:Int, y:Int):Int{
    return x+y
}

参数

1、默认参数 ： 默认参数被省略后，会自动使用默认值
fun sum(x:Int = 0 , y:Int):Int{
    return x+y
}
sum(y=2) // 默认x=0 进行运算 , 这种 “参数名=参数值” 的调用使用了命名参数，避免了参数过多时，无法轻易的分辨出每个参数的具体含义；

2、可变参数 vararg , 在kotlin中 ，可变参数实际通过数组来处理， 特别的，在kotlin中操作数组时，需要使用伸展操作符（*）
fun sum(vararg paras:Int):Int{
    var result:Int =0
    for(para in paras) result+=para
    return result
}
val paraArray = intArrayOf(1,2,3,4)
sum(*paraArray)

单表达式函数

即函数返回了单个的表达式，如：
fun sum(x:Int = 0 , y:Int):Int{
    return x+y
}
等价于 fun sum(x:Int = 0 , y:Int):Int = x+y
等价于 fun sum(x:Int = 0 , y:Int) = x+y // x,y类型已经声明，那么返回类型可推导

成员函数

定义在类或对象的内部的函数

局部函数

定义在函数内部的函数，可以访问其外部函数(即闭包)的成员变量，使用时也要由外部函数来调用

// 成员函数
fun outMethod(x: Int, y: Int) {
    val memberVal = 10
    // 局部函数
    fun innerMethod(z: Double): Double {
        return z * memberVal
    }
}

匿名函数

没有函数名
fun(x:Int = 0 , y:Int):Int{
    return x+y
}

Lambda 表达式

本质上是一个匿名函数，而且可作为一个接口被实现
val sum = {x:Int ,y:Int -> x+y }

高阶函数

将函数作为参数或者返回值的函数
fun sum(x:Int = 0 , y:Int, paraFunc:(a:Int,b:Int)->Boolean):Int{
    return if(paraFunc(x,y)) x+y else x-y
}

在Kotlin中，如果函数最后一个参数接收函数，那么这个函数可以放在外面
fun sum(x:Int = 0 , y:Int, paraFunc:(a:Int,b:Int)->Boolean):Int{
    return if(paraFunc(x,y)) x+y else x-y
}

Kotlin基础（一）

Kotlin基础–常用语法

Kotlin简介

由JetBrains开发的一款基于JVM的编程语言，能够编译成Java字节码，适用于现在所有的Java环境，也可以编译成JavaScript，方便在没有JVM的设备上运行。是目前Android的官方开发语言

官网

Kotlin基础语法

1、Kotlin文件以 .KT 为后缀

2、包的声明和导入与Java相同

3、变量的声明

var 变量名 : 变量的类型 = 变量的初始值  
变量的类型通常可以省略(系统推断)，
但是在没有指定变量的初始值时，不能省略,例如 :*************有待确认，是常量需要还是变量
var  c: Int 不能写成 var c

4、常量的声明

val 常量名 : 常量的类型 = 常量的初始化值

5、函数的定义（与java不同）

无返回值类型
fun sum(a:Int){} 或者 fun sum(a:Int):Unit{}

有返回值类型 
fun sum(a:Int,b:Int):Int{
    return a+b 
}
返回值类型可以自动推断，所以可以省略， 但是被public 修饰的方法，不能省略返回值类型

可变长参数函数 vararg 关键字修饰
fun vars(vararg v:Int){
    for(value in v){print(value)}
}

lambda(匿名函数)
val sumLambda:(Int,Int) ->Int = {x,y -> x+y}
大括号中定义函数的执行逻辑

6、NULL检查

?  指定可以为空 , 如果为空，返回null
定义时： var para:String?="123"
使用时： val result = para?.toInt()

!! 指定不可以为空，如果给定了空值，抛出异常
使用时： val result = para!!.toInt()

7、区间

..       1..5 即 1,2,3,4,5
downTo     5 downTo 1 即 5,4,3,2,1 不能写成 5..1的形式
step    步长  1..5 setp 2 即 1，3，5
until    半包结构 1 until 5 即 [1,5)

基本数据类型

note : 不支持8进制

1、数字的定义支持下划线

val value = 1_0000_0000

2、比较

=== 表示 对比 地址值 ， == 表示对比 值的大小

char 类型不能直接和 int 类型比较

3、隐式转换要注意

和java不同，较小的数据类型不能直接赋值给教大的数据类型

条件控制

1、 if…else

if...else... 可以写成表达式的形式  val max = if(a>b) a else b

2、 when{..else..}

类似与switch操作，从上到下进行判断操作 ，else 就类似与 default

when(x){
    0 -> print(结果为0)
    1 -> print(结果为1)
    else->{
        print(结果异常)
    }
}
